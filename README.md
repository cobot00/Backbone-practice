Backbone-practice
==================

Backbone.jsの手習い用のサンプルコードになります。  
Model、Viewの基本的な使い方を習得することを目的としており、  
サーバーなしで動作するように記述しています。  
index.htmlをブラウザで実行してください。  
  
* **backbone_models.js**  
  Model、Viewを定義しています。  
  
* **backbone_practice.js**  
  backbone_models.jsで定義したModel、Viewのインスタンスを生成しています。  
  
### 依存ライブラリ
* backbone.js  
* jquery.js  
* json2.js  
* underscore.js  
* bootstrap.css  
* bootstrap-responsive.css
